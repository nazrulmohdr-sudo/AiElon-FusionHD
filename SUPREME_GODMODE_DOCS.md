## Troubleshooting

### Chain Already Locked

If you try to add blocks after locking:
```
RuntimeError: Chain is locked. Cannot add new blocks.
```

**Solution**: Add all necessary blocks before locking.

### Chain Not Locked Before Sealing

If you try to seal before locking:
```
RuntimeError: Chain must be locked before sealing.
```

**Solution**: Call `apply_immutable_lock()` before `apply_eternal_seal()`.

### Percentage Logic Value Out of Range

If you provide invalid percentage values:
```
ValueError: Value must be between 0.0 and 1.0
```

**Solution**: Ensure values are in the range [0.0, 1.0].
