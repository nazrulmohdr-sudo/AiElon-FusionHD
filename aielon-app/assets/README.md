# Assets Folder

This folder contains all static assets for the AiElon FusionHD app.

## Required Files:
- **icon.png**: App icon (1024x1024)
- **splash.png**: Splash screen image
- **adaptive-icon.png**: Android adaptive icon (foreground)
- **favicon.png**: Web favicon

## Additional Assets:
- Images
- Fonts
- Videos
- Other media files

Please add your app assets here following Expo's asset guidelines.
